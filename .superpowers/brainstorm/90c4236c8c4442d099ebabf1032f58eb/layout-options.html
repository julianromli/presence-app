Rekomendasi Saya

A. Mission Control paling pas untuk kebutuhan Anda karena `/dev` akan dipakai sebagai control room internal dengan akses mutasi penuh. Pola ini menjaga overview global tetap di atas, tapi masih cepat masuk ke detail workspace dan user tanpa terasa seperti sekadar halaman admin bertab biasa.
